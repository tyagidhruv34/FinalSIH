# **App Name**: Aapda Guide

## Core Features:

- Alert Aggregation: Collect and display disaster alerts from various official sources using a tool, ensuring users have the latest information. The LLM can decide which data sources provide the most pertinent data given a users preferences, as set in the settings.
- Status Sharing: Allow users to post their status updates and share information with others in their vicinity. The LLM can check for duplicate submissions to keep a consistent tone. 
- Resource Locator: Locate nearby resources like shelters, hospitals, and help centers, displayed on an interactive map.
- Emergency Contacts: Maintain a list of important emergency contacts, including local authorities and helplines.
- Offline Access: Provide access to essential information even without an internet connection, such as basic safety guidelines and emergency contacts.
- Multilingual Support: Offer content in multiple languages to cater to diverse user groups.

## Style Guidelines:

- Primary color: A calm and trustworthy blue (#4A86E8) to instill confidence and reliability in users during crises.
- Background color: Light grey (#F0F4F8), almost white, to ensure readability and minimize visual clutter.
- Accent color: A contrasting orange (#FF7043) to highlight critical information, alerts, and calls to action, clearly differentiating them from the primary content.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear readability and a modern yet approachable feel.
- Use clear, universal icons for key functions such as alerts, locations, and contacts.
- Maintain a clean, intuitive layout to facilitate quick access to critical information.
- Use subtle animations for map transitions and loading alerts to enhance the user experience without being distracting.
